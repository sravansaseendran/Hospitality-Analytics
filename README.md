# Hospitality-Analytics
Developed an interactive Hospitality Analytics Dashboard to analyze hotel performance and support data-driven decision-making across multiple properties and cities. The project focuses on transforming raw booking and revenue data into meaningful business insights.
Analyzed overall business performance using key hospitality KPIs such as revenue trends, booking success, occupancy, cancellations, and customer ratings
Built city-wise and property-wise revenue analysis to compare performance across locations
Performed room category and class-wise analysis to understand demand patterns and customer preferences
Evaluated booking behavior across multiple platforms to identify dominant booking channels
Analyzed weekday vs weekend performance to understand demand variation
Created time-based analysis to track monthly and weekly booking and revenue trends
Designed an interactive and user-friendly dashboard with slicers for dynamic filtering and drill-down analysis
