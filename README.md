![p1](https://github.com/user-attachments/assets/02c0fb97-1dbc-4aaa-b6c3-4e582aabc6ce)# Hospitality-Analytics
Developed an interactive Hospitality Analytics Dashboard to analyze hotel performance and support data-driven decision-making across multiple properties and cities. The project focuses on transforming raw booking and revenue data into meaningful business insights.

Tools & Technologies
- Excel (Data cleaning & analysis)
- Power BI (Dashboard & visualization)
- SQL (Data querying)
- Basic statistics & KPI analysis

 Key Analysis Areas
- Revenue and booking trends
- Occupancy performance
- City and property level analysis
- Weekly and monthly performance trends

Key Outcomes
- Built an interactive Power BI dashboard
- Identified performance patterns and business insights
- Improved data-driven reporting approach
